Report Information
ISD Reimbursement Form
  • One row per expense item
  • Grouped by currency (HKD, RMB, Others)
  • Sequential receipt numbering
  • Receipt attachment status
Financial Expense Form
  • One row per claim
  • Combined item descriptions
  • Grouped expense categories
  • Unique claim identifiers
Receipts Export
  • ZIP file with all receipts
  • Files named with claim UUIDs
  • Organized by date range
  • Preserves original formats
How to Use Reports
For Accounting Department:
  1. Generate Financial Expense Form for claim summaries
  2. Use the UUID to cross-reference with detailed records
  3. Export receipts ZIP for documentation
  4. Import CSV data into accounting software
For Employee Reimbursement:
  1. Generate ISD Reimbursement Form for submissions
  2. Print or email the CSV to HR/Finance
  3. Include receipt ZIP as supporting documentation
  4. Keep claim UUIDs for tracking purposes
